Variabler i JavaScript

I JavaScript bruker vi variabler for å lagre og manipulere data. Det finnes tre måter å deklarere variabler på: var, let og const. Forskjellen mellom dem handler hovedsakelig om scope (område hvor variabelen er gyldig), mutabilitet (om verdien kan endres) og hoisting (hvordan JavaScript håndterer variabler før koden kjøres).

Var - funksjons-scope og hoisting

Var kan brukes før den er deklarert pga. hoisting, men verdien blir undefined inntil den er tilordnet.

Let - Blokk-scope og ingen re-deklarering

let er blokkområdet, så den eksisterer bare i { } der den er definert.

Const - Konstant verdi og blokk-scope

const kan ikke endres, men objekter/arrayer kan endres innholdsmessig.

Funksjoner

En callback i JavaScript er en funksjon som sendes som et argument til en annen funksjon og kjøres senere, ofte etter at en asynkron operasjon er fullført. Dette brukes mye i webutvikling for å håndtere hendelser, AJAX-forespørsler, animasjoner og mer.
