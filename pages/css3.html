Teori

CSS-egenskaper for Flexbox

Flexbox og Grid er beslektede teknikker som brukes til å styre hvordan elementer plasseres i forhold til hverandre på en nettside.

Flexbox bruker vi gjerne når vi skal plassere elementene i én dimensjon – altså ved siden av hverandre eller under hverandre.

Grid brukes ofte når vi skal plassere elementer både ved siden av og under hverandre i et todimensjonalt rutenett.

En flexbox inneholder flex-items. Når et HTML-element settes til å være en flexbox, kan vi kontrollere hvordan elementene inni dette elementet skal utformes, og hvordan de skal plasseres i forhold til hverandre.

Vi bruker CSS-egenskapen display: flex; for å si at et HTML-element skal være en flexbox. Alle elementene som ligger inni en flexbox blir automatisk flex-items.

Eksempel (forenklet):
Sett en container til display: flex;, og plasser tre bokser inni – disse blir da flex-items som du kan styre med flere egenskaper som justify-content, align-items og flex.
Teori

Justify-content og align-items

Egenskapen justify-content brukes til å styre hvordan flex-items fordeles langs hovedaksen i flexcontaineren (for eksempel horisontalt hvis raden går fra venstre til høyre).

Egenskapen align-items brukes til å justere hvordan elementene plasseres på tvers av hovedaksen. Vi unngår ofte å si «vertikalt» og «horisontalt» i Flexbox-sammenheng, fordi retningen kan endres med flex-direction.

Så lenge elementene ligger på rad etter hverandre, kan vi bruke align-items til å justere om elementene:

  • strekker seg ut over hele høyden
  • ligger i toppen
  • ligger på midten
  • ligger nederst
CSS Resultat for flex-items
align-items: stretch; (standard) Elementene trekker seg over hele høyden
align-items: flex-start; Elementene plasseres i toppen
align-items: center; Elementene plasseres på midten
align-items: flex-end; Elementene plasseres i bunnen
Husk: Hvis du endrer flex-direction til column, vil «retningen» til justify-content og align-items byttes om.

align-items: flex-start

align-items: flex-start

align-items: flex-end

Teori

Hvordan elementene i en flexbox strekker seg ut

En viktig egenskap ved Flexbox er at vi kan styre hvordan elementene inni flexboxen fordeler plassen mellom seg.

Husk at elementene som ligger inni en flexbox kalles flex-items. På disse kan vi sette CSS-egenskapen flex med verdier som avgjør størrelsesforholdet mellom elementene.

Hvis disse elementene har lik flex-verdi, blir de like store. Ulike verdier gir ulike forhold, for eksempel: 

.elem1 { flex: 1; }
.elem2 { flex: 2; }
.elem3 { flex: 7; }

I dette tilfellet vil .elem3 ta mest plass, siden den har høyest flex-verdi.

Teori

Midtstille elementer med Flexbox

Å midtstille innhold på nettsider er ofte et estetisk og praktisk ønske. Med Flexbox har vi en enkel oppskrift for å plassere innholdet helt i midten:

  1. Definer et element som en flexbox med display: flex;.
  2. Sett align-items: center;.
  3. Sett justify-content: center;.
  4. Det du legger i flexboxen vil nå være midstilt.
  5. Hvis du skal midstille noe som består av flere elementer, kan du samle dem i et eget HTML-element (for eksempel en <div>) som du midtstiller.
Ekstra tips: hvis du vil midtstille teksten inni et element, kan du i tillegg bruke text-align: center; på det elementet.

Midtstille elementer

Teori

Flex-direction: column og menyer

Mange nettsider har en meny (navigasjon) som inneholder lenker til andre sider. Denne ligger ofte inni et <nav>-element.

Med Flexbox kan vi enkelt styre hvordan menyen skal se ut – for eksempel om lenkene skal ligge ved siden av hverandre eller under hverandre.

Hvis vi vil strekke menypunktene over hele bredden, kan vi gi hvert menypunkt flex: 1;. For å legge menypunktene under hverandre, kan vi bruke: 

.meny {
  display: flex;
  flex-direction: column;
}

Da vil hvert meny-element legge seg under det forrige, fortsatt kontrollert av de samme Flexbox-prinsippene.

Teori

CSS Grid – Neste steg

I moderne webutvikling er det en stor fordel å bruke grid og flexbox sammen for å lage fleksible og responsive nettsider. En responsiv nettside tilpasser HTML-elementene til forskjellige skjermstørrelser – fra mobil til store skjermer.

Kildekode til eksemplene i denne delen finner du her: https://github.com/Kfmoss/GRID-uke45.git .

Hva er et grid?

Et grid-layout lager et rutenett der HTML-elementer kan ligge både ved siden av hverandre og over/under hverandre. Rutene i et grid er fleksible – innhold kan strekke seg over flere ruter.

Et HTML-element som utformes som et rutenett kalles et grid, og elementene inni kalles grid-items.

Husk: Flexbox er ofte best når du skal styre én retning (rad eller kolonne), mens Grid er perfekt når du vil styre både rader og kolonner samtidig.

Aktivere Grid med display: grid;

For at et element skal bli et grid, må du gi det display: grid; i CSS. Først da vil egenskaper som grid-template-columns og grid-template-rows ha effekt. 

.maincontainer {
display: grid;
}

Definere kolonner og rader

Et rutenett består av kolonner og rader:

  • grid-template-columns – styrer bredden på kolonnene.
  • grid-template-rows – styrer høyden på radene.

En måte å definere et grid på, er å liste opp flere lengdeverdier. Disse verdiene tilsvarer bredden på kolonnene (eller høyden på radene). 

.maincontainer {
display: grid;
width: 100%;
height: 100vh; /* tar hele høyden til vinduet */
grid-template-columns: 1fr 2fr 1fr;
grid-template-rows: 30% 70%;
}

I eksemplet over:

  • 1fr 2fr 1fr deler containeren i tre kolonner, der den midterste er dobbelt så bred som de to på sidene.
  • 30% 70% gjør at første rad får 30 % av høyden, og andre rad 70 %. Når du bruker prosent, er høyden alltid proporsjonal med størrelsen på containeren. :contentReference[oaicite:4]{index=4}
Begreper: fr betyr fractional unit (en del av tilgjengelig plass). Prosent (%) er alltid relativt til størrelsen på grid-containeren.

Grid-items og områder

Hvert element inni grid-containeren blir et grid-item. Med CSS kan vi bestemme hvilke ruter hvert element skal ligge i, og hvor mange ruter det skal strekke seg over. 

<div class="maincontainer">
<section class="sec0">...</section>
<section class="sec1">...</section>
<section class="sec2">...</section>
<section class="sec3">...</section>
<section class="sec4">...</section>
<section class="sec5">...</section>
</div>
.sec0 { display: flex; background-color: cadetblue; }
.sec1 { display: flex; background-color: coral; }
.sec2 { display: flex; background-color: cornflowerblue; }
.sec3 { display: flex; background-color: rgb(223, 10, 205); }
.sec4 { display: flex; background-color: rgb(181, 235, 190); }
.sec5 { display: flex; background-color: rgb(143, 133, 125); }

Hvert section-element blir et grid-item. I eksempelet i PDF-en brukes både Grid og Flexbox: Grid til å lage selve rutenettet, og Flexbox inni rutene for å styre innholdet. :contentReference[oaicite:5]{index=5}

Grid i nettaviser

Mange nettaviser bruker Grid for å utforme forsiden. Hver nyhetssak er en lenke som inneholder bilde, overskrift og eventuell ingress. Grid brukes til å styre hvor mye plass hver sak tar i bredden og høyden. :contentReference[oaicite:6]{index=6}

Vi kan definere klasser som sier hvor mange kolonner og rader en sak skal strekke seg over: 

.one-column  { grid-column: span 1; }
.two-columns { grid-column: span 2; }
.three-columns { grid-column: span 3; }

/* tilsvarende for rader */
.one-row   { grid-row: span 1; }
.two-rows  { grid-row: span 2; }
.three-rows { grid-row: span 3; }

På denne måten kan forsiden bygges opp som et rutenett der noen saker får mer plass (for eksempel toppsaker), mens andre blir mindre.

Tekst nederst og bilde i bakgrunnen

I eksemplet med nettaviser gjenstår det å:

  • flytte teksten ned til venstre i hver rute
  • legge et bilde bak teksten

Dette kan løses ved å definere hver rute som et flex-item, og flytte teksten ned med align-items: flex-end; og litt padding. Bildet kan legges inn enten som bakgrunnsbilde eller et vanlig <img>-element som plasseres bak teksten med absolutt posisjonering. :contentReference[oaicite:7]{index=7} 

.news-item {
display: flex;              /* bruker flex inni grid-ruta */
align-items: flex-end;      /* tekst ned i bunnen */
padding-left: 1rem;
padding-bottom: 1rem;
background-size: cover;
background-position: center;
}
Oppsummert: Bruk Grid til å lage rutenettet (layouten) for siden, og Flexbox inni rutene for å plassere tekst og annet innhold på en fleksibel måte.
